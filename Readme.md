# TFM MLOps Apple

Proyecto de despliegue y monitorización en Azure Machine Learning
de un modelo supervisado de clasificación de sentimientos.

## Clases

- negative
- neutral
- positive

## Modelo

Pipeline de Scikit-learn compuesto por:

- limpieza de texto
- TF-IDF
- regresión logística

## Fuentes de datos

- Cuenta oficial de Apple en X
- Google News RSS
